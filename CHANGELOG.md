# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added
- Initial release
- Telegram bot for remote Claude Code control
- `/task` command to create tasks from phone
- `/status` command for real-time monitoring
- `/approve` and `/reject` for approval management
- User ID authentication
- File-based approval system
- Comprehensive documentation
- Security audit completed

### Security
- User authentication via Telegram ID
- Environment variable configuration
- No exposed network ports
- Audit logging for all actions
